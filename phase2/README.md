# Phase 2 – Plagiarism Detection

## Overview

Phase 2 extends the preprocessing pipeline developed in Phase 1 by introducing the core plagiarism detection logic. This phase focuses on comparing tokenized source code submissions and identifying similarities between programs using modular detection components.

---

## Directory Structure

```text
phase2/
├── Makefile
├── main.cpp
├── plagiarism_checker.cpp
├── plagiarism_checker.hpp
├── tokenizer.cpp
├── structures.hpp
├── ainur/
└── hogwarts/
```

---

## Components

| File | Description |
|------|-------------|
| `main.cpp` | Entry point for the Phase 2 implementation |
| `plagiarism_checker.cpp` | Implements the plagiarism detection logic |
| `plagiarism_checker.hpp` | Interface for the plagiarism checker |
| `tokenizer.cpp` | Performs source code tokenization and preprocessing |
| `structures.hpp` | Defines common data structures used throughout the implementation |
| `ainur/` | Sample submissions and test data |
| `hogwarts/` | Additional sample submissions and test data |

---

## Building

Compile the project using the provided Makefile.

```bash
make
```

---

## Running

Execute the binary generated by the Makefile.

```bash
./main
```

---

## Objective

The objective of this phase is to compare preprocessed source code submissions and identify potential plagiarism by analyzing similarities between tokenized programs.

---

## Progression

This phase builds upon the tokenizer developed in **Phase 1** and introduces the core comparison framework. The implementation is further extended in **Phase 3**, which integrates multiple detection modules into the final plagiarism detection system.
