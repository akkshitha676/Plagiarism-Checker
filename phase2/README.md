# Phase 2 - Plagiarism Detection

## Overview

Phase 2 extends the tokenizer developed in Phase 1 by implementing the core plagiarism detection logic. It introduces mechanisms for comparing source code submissions and evaluating similarities between programs.

## Directory Structure

```
phase2/
├── main.cpp
├── plagiarism_checker.cpp
├── plagiarism_checker.hpp
├── tokenizer.cpp
├── structures.hpp
├── Makefile
├── ainur/
└── hogwarts/
```

## Test Data

The `ainur` and `hogwarts` directories contain sample submissions and test cases used during development and evaluation.

## Build

```bash
make
```

## Run

```bash
./main
```

The executable name depends on the generated build output.
