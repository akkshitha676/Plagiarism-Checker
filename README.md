# Plagiarism Checker

A C++-based plagiarism detection system developed as part of the **CS293 – Data Structures and Algorithms Laboratory** at **IIT Bombay**. The project explores techniques for analyzing and comparing C++ source code submissions by progressively building a plagiarism detection pipeline across multiple development phases.

The repository demonstrates the evolution of the system from source code tokenization and preprocessing to a modular plagiarism detection framework.

---

## Features

- Tokenization of C++ source code
- Source code preprocessing and normalization
- Multiple plagiarism detection modules
- Modular and extensible project architecture
- Docker support for a reproducible development environment
- Incremental implementation across three project phases

├── phase2/

### Phase 1

Implements the foundational components required for source code preprocessing, including tokenization and validation using sample test cases.

### Phase 2

Builds upon the preprocessing pipeline by introducing plagiarism detection logic and mechanisms for comparing source code submissions.

### Phase 3

Integrates the complete plagiarism detection pipeline with multiple detection modules into the final implementation.

## Building

Each phase contains its own `Makefile`.

Navigate to the desired phase and build the project:

```bash
cd phase3
make
```

---

## Running

After building, execute the binary generated by the corresponding `Makefile`.

Example:

```bash
./main
```
