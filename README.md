# Phase 3 – Final Plagiarism Detection System

## Overview

Phase 3 contains the final implementation of the plagiarism detection system. It combines source code preprocessing with multiple plagiarism detection modules to form a complete comparison pipeline for analyzing C++ program submissions.

This phase builds upon the tokenizer introduced in Phase 1 and the comparison framework developed in Phase 2, integrating them into a modular and extensible architecture.

## Design

The plagiarism detection pipeline consists of the following stages:

1. Tokenize the source code.
2. Normalize the token stream.
3. Apply multiple plagiarism detection modules.
4. Aggregate the results to evaluate similarity between submissions.

The modular design allows individual detection strategies to be developed and evaluated independently.
